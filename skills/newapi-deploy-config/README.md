# New API 部署与模型配置（Host 网络）

## 简介

该技能用于在本机通过 Docker（host 网络）部署 New API，并提供后续通过管理 API 配置模型渠道的流程模板（不包含任何密钥）。

## 适用场景

- 需要在当前机器快速部署 New API
- 需要宿主机直连容器（host 网络模式）
- 国内网络拉取镜像困难，需要为 Docker 守护进程配置代理
- 需要通过管理 API 批量配置模型渠道

## 内容要点

- Docker 安装与代理配置（可选）
- 镜像拉取与容器启动（host 网络）
- 服务验证与初始化管理员
- 通过管理 API 创建渠道（模板化）
- 安全注意事项：不要写入/提交任何密钥
