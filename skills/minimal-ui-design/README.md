# Minimal UI Design（极简 UI 设计）

## 简介

该技能用于在保持可理解性的前提下，打造低噪声、图标优先的极简界面，适合精简文本、合并控制项和简化操作面板。

## 适用场景

- 需要极简、干净、低噪的 UI
- 正在从文本按钮迁移到图标按钮
- 需要在简化后保持新手可用性
- 精简搜索、工具栏或操作栏

## 内容要点

- “减少而非隐藏”的信息组织原则
- 图标优先但需可访问性标签
- 低干扰引导（占位符 / tooltip / keycap）
- 密度与可读性的平衡
- 极简 UI 的常见错误与修正
