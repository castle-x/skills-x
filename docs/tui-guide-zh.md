```
安装: 1 | 更新: 1 | 卸载: 1
────────────────────────────────────────────────────────────
Space 选择  f 收藏  u 检测更新  A 全选  Enter 确定  b 返回  q 退出
```

**选择操作**

| 按键 | 行为 |
|------|------|
| `Space` | 切换：未安装 → `[+]安装` · 已安装 → `[-]卸载` |
| `A` | 循环：全部安装/更新 → 全部卸载 → 重置 |
| `Enter` | 确认并进入安装流程 |

**搜索与筛选**

| 按键 | 行为 |
|------|------|
| `/` 或 `、` | 进入搜索模式，输入 skill 名称 |
| `#` | 打开标签选择器，`↑`/`↓` 切换，`Enter` 应用 |
| `Esc` | 退出搜索 / 取消标签选择器 |

可用标签：`#starred` `#featured` `#ai-efficiency` `#planning` `#web-frontend` `#mobile` `#backend` `#testing` `#code-review` `#office` `#design` `#writing` `#media` `#skills`

**技能操作**

| 按键 | 行为 |
|------|------|
| `f` | 收藏 / 取消收藏当前技能 — 星标技能始终排在列表最前，保存于 `~/.config/skills-x/starred.json` |
| `u` | 检测当前已安装技能是否有新版本 — 在状态区显示版本对比信息 |
